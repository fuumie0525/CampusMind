# CampusMind 校园服务演示知识

> 本文件仅用于演示 RAG 检索，不代表具体学校的正式政策。

## 校园网认证失败

1. 确认统一身份认证账号可正常登录。
2. 忘记当前网络后重新连接，并重新打开认证页面。
3. 检查系统时间、代理、VPN 和自定义 DNS。
4. 记录楼栋区域、发生时间、设备系统和错误码。
5. 完成自查仍失败时联系学校网络中心。

## VPN 连接超时

- 使用学校官网下载的当前版本客户端或配置文件。
- 检查系统时间和统一身份认证状态。
- 暂时关闭冲突的代理软件后重试。
- 不要公开发送密码、验证码或完整身份信息。

## 校园卡遗失

- 立即通过学校官方 App、公众号、自助终端或一卡通中心挂失。
- 按学校要求携带有效证件办理补卡。
- 新卡启用后确认余额和身份信息同步情况。
- CampusMind 只提供流程说明，不会执行真实挂失或补卡。

## 宿舍报修

- 在后勤报修平台填写楼栋、房间、故障类型并上传照片。
- 保存报修单号，便于查询进度。
- 出现漏电、冒烟、火情或大面积漏水时，先远离危险区域并联系宿管、保卫处或紧急服务。
